CSS gradient

เป็นโค้ดที่เอาไว้เป็นแบ็คกราว header และไล่สี ต้องมีสองสีเป็นอย่างต่ำมีอยู่สามแบบ

อ้างอิง: https://www.youtube.com/

  • Linear Gradients (การไล่สีที่เป็นขึ้น,ลง,ซ้าย,ขวาและแนวทแยงมุมได้)

    • Syntax: background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

  • Radial Gradients (การไล่สีเริ่มที่ตรงกลาง เป็นได้ทั้งทรงกลมหรือวงรี)

    • Syntax: background-image: radial-gradient(shape, color-stop1, color-stop2, ...);

  • Conic Gradients (เป็นการไล่สีที่หมุนวนตรงกลาง สามารถปรับองศาได้)

    • Syntax: background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...);

CSS box-shadow

เป็นโค้ดที่เอาไว้ทำเงาตรงกล่องไดอารี่ที่ทำ

Syntax: box-shadow: none|h-offset v-offset blur spread color |inset|initial|inherit;

  • none: ไม่ปรากฏเงา
  • ประเภทที่ใช้ร่วมกัน
    1. h-offset: เคลื่อนเงาไปทางขวากล่อง
    2. v-offset: เคลื่อนเงาไปข้างล่างกล่อง
    3. blur: ทำให้จาง
    4. blur: ทำให้กระจาย
    5. color: สีของเงา
  • inset: เปลี่ยนเงาด้านนอกกล่องเป็นด้านในกล่อง
  • initial: จะเป็นการเซท default
  • inherit: ถ้าตามหลัง property ตัวไหน มันก็จะสืบทอดค่ามา
