Diary 5 : รีแอครีใจ

16/11/2565

วันนี้ลองเรียนรู้ React ที่เป็น JavaScript library จากรุ่นพี่ ดูแล้วรู้สึกว่ามันยากตอนที่ฟังรุ่นพี่พูดมา แต่พอลองค้นคว้าอะไรเองก็พอจับทางได้ง่ายขึ้น อาจจะเป็นเพราะการเรียนออนไลน์หลายๆครั้งที่ไม่เข้าใจแล้วเรียนเองจนชิน แต่ไม่ได้แปลว่าตอนนี้เข้าใจทุกอย่าง โดยเฉพาะ syntax ที่ยังไม่ชิน เพราะฉะนั้นจะลองฝึกเรื่อยๆ เพราะรุ่นพี่บอกว่าถ้าเขียนได้ก็เขียนเว็บได้ทุกรูปแบบแล้วล่ะ
